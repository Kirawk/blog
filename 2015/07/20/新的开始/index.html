如何选书？

首先对于花时间去选书，我觉得是很值得的。看似浪费时间，实则为我们节约更多时间与精力。所谓“磨刀不误砍柴工”，说得就是这个理。
对于选书，哪些才是好书呢？我个人觉得一定要符合你当前所处的年纪段，同时还得评估下自己目前所处的水平。举个例子来说，刚学java(以前没接触过)，很多人都觉得《Think in java》不错！的确，这属于一本经典之作，可要是对于入门选手来说，那就未必能读的下去了！相反，《Head First in java 》就很适合入门书。举这个例子，就是为了说明选书前一定要找好自己的位置！不可否认，有些牛人是可以一步跨到较高层次的。我绝得：对于普通人这样一步一步来可以在取得最大进步的同时避免因打击太大而丧失信心
找好自己的位置后，就可以进入下一步了。我一般喜欢在问答类的网站：比如知乎。去看下应该选择哪些书？（适合自己当前水平）一般会有人推荐的。在此基础上，我会选择一批书单出来。以此书单出发，去豆瓣读书!注意下面几点:
1.看书籍的评分多少，有多少人评论。
2.看下面的书评，对于好评与坏评做到自己心中有数就行，不必太纠结。有时可以看到很多精彩的书评，同时我觉得自己可以在看完书后写下书评，和这里的人对比下下，让不同人的思想得到碰撞!
3.豆瓣待得时间长了，可以找到很多好的豆列！当然还可以使用谷歌搜索。如：
教育学site:book.douban.com/doulist/
当然你也可以打开豆瓣该书的目录好好看看。通过目录，看能否发现自己需要的东西。接下来可以加入购物车，去京东或亚马逊下单。下单前，你还可以再去看下亚马逊或京东等这内购物网站下评价！无论是书的内容还是书本身的质量如何！
如果是在实体店你买书或图书馆借书，我觉得可以从下面着手：
1.打开书后，看书中的印刷的次数，第几版了。一般经典的书籍会再版很多次，因此通过这个可以找到好书。
2.看这本书的作者，一般你找到好书后，那么这书的作者的其他书籍也可以找来读一读。前提是你手中已经积累了好作者的名单。
3.看出版社，经常看书发现有些出版社的书总是值得一读的。
最后来说下我最喜欢的一种找书方法“顺藤摸瓜”。所谓顺藤摸瓜：就是以你当前手中的书为起点，将书中出现的书籍、作者等信息记录下来。看完书后，将书中所提到的书籍或引用到的书籍、作者整理下。然后搜集整理书单。这种方法的好处的有：
1.若你看的书的作者是个牛人，那他引用或推荐的绝对也是牛人（或许是骨灰级的！）这样你就能花较少的时间找到优秀的书。
2.通过他们的眼睛，我们可以过滤掉一大批没什么干货的书籍。这就像我们之所以能够看得很远，那是因为站在了巨人的肩膀上！
以上就是目前自己找书的一些小感悟！
